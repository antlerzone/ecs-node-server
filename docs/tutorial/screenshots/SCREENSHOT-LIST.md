# Screenshot 檔名清單 — 截圖後請用這個名字存檔

把截圖存進 **`docs/tutorial/screenshots/`** 資料夾，檔名照下面列表。  
教程裡已經寫好這些檔名，你只要 **截圖 → 用下面名字重新命名 → 放進此資料夾** 就會自動顯示。

**建議到 demo 站截圖：** **demo.colivingjb.com** 沒有接後端，已內建示範資料（agreement、tenancy、invoice、report、expenses、admin 等都有 sample items）。用 demo123 / 登入後選 **Owner** 或 **Tenant** 或 **Operator**，各頁面都會有資料，方便直接截圖對應本清單。

支援副檔名：**.png** 或 **.jpg**（教程裡用 .png，你存 .jpg 的話把下面表格的 .png 改成 .jpg 即可）。

## 使用方式

1. 依上表 **到對應頁面截圖**。  
2. 把檔案 **重新命名** 成表格裡的檔名（例如 `owner-01-login.png`）。  
3. 放進 **`docs/tutorial/screenshots/`**。  
4. 不用改教程 .md，裡面已經掛好這些檔名，放進去就會顯示。

若用 **.jpg**：把檔名從 `.png` 改成 `.jpg`，並到對應的 `owner-tutorial.md` / `tenant-tutorial.md` / `operator-tutorial.md` 把該圖的副檔名從 `.png` 改成 `.jpg`（或全專案統一用 .png 較簡單）。
